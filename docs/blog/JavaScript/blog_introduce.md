#### 博客简介

这个网站是利用docsify技术栈搭建的文档网站，后面想通过学习node、express、vue等技术栈，利用所学的知识实现一个博客系统，于是开始实施这个博客。

#### 博客技术栈

后端：node、express、mongodb

admin端：vue、vuex、vue-router、wangeditor、mavon-editor、element-ui、axios

用户端：vue、element-ui、axios
